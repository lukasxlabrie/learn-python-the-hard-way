* **argv**: List of command-line arguments (`from sys import argv`).
* **script, filename = argv**: Unpacks script name and filename from command line.
* **open(filename)**: Opens the file for reading.
* **read()**: Reads the entire contents of a file.
* **print()**: Displays output to the console.
* **input()**: Waits for and returns user input.
* **prompt**: Custom string shown before `input()` (e.g., `"> "`).
* **f-strings**: Format strings to include variables, e.g., `f"File: {filename}"`.
